## 모듈

```
- 스크립트 제작 후 길어진 파일의 간편한 유지를 위해 여러 개의 파일로 나눔 
- 이 때 각 나눠진 파일들에 포함된 정의들을 복사하지 않고 다른 파일에서 사용하기 위함
- 정의들을 파일에 넣고, 스크립트나 인터프리터 대화형 모드에서 사용하는 방법
- 한 모듈의 정의들이 다른 모듈이나 메인 모듈로 import될 수 있음
- 모듈은 다른 모듈들을 import할 수 있다. 
- 메인 모듈: 최상위 수준에서 실행되는 스크립트에서 액세스하는 변수들의 컬렉션
- 모듈 : 파이썬 정의와 문장을 담고 있는 파일
```

```python
#모듈 사용 예시 
#fibo.py 파일에 def fib():로 피보나치 함수를 만들어 놓는다면, 다른파일에서

import fibo
fibo.fib(n) # 등으로 fibo.py의 fibo.py 의 fib()함수를 사용가능하다. 

# 자주 사용한다면 지역이름으로 대입가능
fib = fibo.fib 
fib(n) #으로 호출 
```

```
#주의# 
효율성의 이유로, 각 모듈은 인터프리터 세션마다 한 번만 임포트됩니다. 그래서, 여러분이 모듈을 수정하면, 인터프리터를 다시 시작시켜야 합니다 --- 또는, 대화형으로 시험하는 모듈이 하나뿐이라면, importlib.reload() 를 사용하세요. 예를 들어,
import importlib; importlib.reload(modulename).

```

### 표준 모듈

- dir() 함수

```
내장함수 dir()을 쓰면 모듈이 정의하는 이름들을 찾는 데 사용된다. 
# 사용법 1)
dir(함수명)  # 문자열들의 정렬된 리스트를 반환

# 사용법 2)
dir()   	# 인자가 없으면 현재 정의한 이름들을 나열 
		    # (모든 형의 이름을 나열함, 변수, 모듈, 함수 등등)

#사용법 3) 
import builtins # 표준 모듈 builtins import 후 
dir(builtins)   # 내장 함수와 변수들의 이름을 보고 싶을 때 사용 	
				# builtins에 내장 함수와 변수 리스트가 정의되어 있음
```
